About

Our app is a simple web application for tracking language learning progress. Users can create an account, log in, and then log their language learning sessions. They can view their past logs and see their progress over time.

Requirements for importing logs

To import logs, users need to provide a CSV file with the following columns:

  • Date (format: DD.MM.YYYY)
  • Duration (in minutes)
  • Language (one of the following: Python, Java, C++, Pascal, HTML:o), JavaScript, MyNewLanguage)
  • Rating (a number from 1 to 5)
  • Notes (optional)

Here's an example CSV file that you can download and use as a template:

Download Example CSV File

Please note that any invalid or missing data will cause the import to fail.

Supported languages

Our app currently supports the following languages:

  • Python
  • Java
  • C++
  • Pascal
  • HTML
  • JavaScript
